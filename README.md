# Sistema de Drones con IA
Sistema de inteligencia artificial que simula la distribución de alimentos mediante drones en zonas afectadas por inundaciones en Córdoba.
Utiliza el algoritmo A* para encontrar rutas óptimas evitando obstáculos y Minimax para la toma de decisiones en un entorno adversario dinámico.
El sistema incluye una interfaz gráfica que permite visualizar el comportamiento del dron y las condiciones del entorno.
